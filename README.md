# qbe-take-home-assignment-03

## System requirementes

- Ubuntu 20.04 or later
- Python 3.8 or later
- Redis 7.0.15

To install Redis, follow [these steps](INSTALL_REDIS.md).

## Setup
Clone this repository. Then, from the root of this local repository, run this:
```
python3 -m venv .venv
. .venv/bin/activate
pip install -e .
```
